# Excel 文本替换工具 - Windows 使用指南

## 一、安装 Java 环境

本工具依赖 Java 运行环境，如果电脑未安装过 Java，请先安装。

### 1. 下载 Java

访问以下网址下载：
- 官方下载：https://www.java.com/zh-CN/download/
- 推荐版本：Java 11 或更高版本（LTS 版本）

### 2. 安装 Java

1. 双击下载的 `jre-xxx.exe` 安装程序
2. 点击「安装」按钮，等待安装完成
3. 安装完成后关闭安装窗口

### 3. 验证安装

按 `Win + R` 键，输入 `cmd` 回车，在命令行中运行：

```cmd
java -version
```

如果看到类似以下输出，说明安装成功：

```
openjdk version "11.0.20" 2023-07-18
OpenJDK Runtime Environment ...
OpenJDK 64-Bit Server VM ...
```

## 二、准备文件

将以下所有文件放到**同一个文件夹**中（例如在桌面新建一个名为 `Excel替换工具` 的文件夹）：

| 文件名 | 说明 |
|--------|------|
| `SoilMoistureHQT.jar` | 主程序文件 |
| `run.bat` | Windows 启动脚本 |
| `config.json` | 替换规则配置文件 |
| `data.xlsx` | 数据源 Excel 文件 |
| `template.txt` | 待替换的文本模板文件 |

**注意事项**：
- 所有文件必须放在**同一目录**下
- 文件名称必须与上方列表一致（区分大小写）
- 不要将文件放在带有中文或空格路径较深的目录中

## 三、运行工具

### 方式一：双击运行（推荐）

直接双击 `run.bat` 文件，程序会自动执行并输出结果。

### 方式二：命令行运行

1. 在文件夹空白处按住 `Shift` + 右键
2. 选择「在此处打开命令窗口」或「在终端中打开」
3. 输入以下命令并回车：

```cmd
run.bat
```

## 四、查看结果

程序运行成功后：

1. 会生成新文件 `template.hqt`（与 `template.txt` 在同一目录）
2. 该文件为替换完成后的最终结果
3. 终端窗口底部会显示「程序执行完毕」

**不要修改原始模板文件**，如需多次运行，原始 `template.txt` 会保持不变，每次运行都会生成新的 `.hqt` 文件。

## 五、常见问题

### 问题 1：双击 `run.bat` 后闪退

原因：Java 环境未安装或配置不正确。

解决方法：
1. 按 `Win + R`，输入 `cmd` 回车
2. 输入 `java -version` 检查 Java 是否安装
3. 如提示"不是内部或外部命令"，请重新安装 Java

### 问题 2：提示「找不到 SoilMoistureHQT.jar」

原因：JAR 文件与 `run.bat` 不在同一目录。

解决方法：确保所有文件放在同一个文件夹中。

### 问题 3：程序运行但替换结果不符合预期

原因：`config.json` 配置或 Excel 数据可能不正确。

解决方法：
1. 检查 `config.json` 中的行号、列号是否正确
2. 检查 `data.xlsx` 中的数据是否与预期一致
3. 确认 `--sheet` 参数指定的工作表名称与 Excel 中的名称一致

### 问题 4：需要修改 Excel 文件或模板

修改 `data.xlsx` 或 `template.txt` 后，直接重新运行 `run.bat` 即可，无需重新编译。

## 六、高级用法

如需自定义参数，可以手动在命令行中运行：

```cmd
java -jar excel-text-replacer-1.0.0.jar --excel 数据文件.xlsx --text 模板文件.txt --config 配置.json --sheet 工作表名称
```

| 参数 | 说明 | 默认值 |
|------|------|--------|
| `--excel` | Excel 数据文件路径 | `data.xlsx` |
| `--text` | 待替换的文本模板 | `template.txt` |
| `--config` | 替换规则 JSON 配置 | `config.json` |
| `--sheet` | 指定 Excel 工作表名称 | 自动检测第一个有数据的工作表 |
